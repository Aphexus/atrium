The main goal of this project is to create a 3D game entirely in D language,
exploring its fitness for real-time application development.
D is a modern compiled language which combines efficiency, control, and modeling power, with safety and programmer productivity.

Learn more about D
Modern graphics
The game is built using DGL - a custom graphics engine on top of OpenGL that supports shader-based effects,
dynamic shadows and infinite number of light sources.

DGL
High level of interactivity
You will be able to walk on any surface and push any object, use special devices to affect gravity and other physical behaviours of the environment.
This part of the game is driven by our custom physics engine - dmech.

dmech
