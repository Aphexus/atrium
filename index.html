Atrium

First person action game with physics based puzzles

The goal of this project is to create a first person shooter game with physics based puzzles, entirely in D language.

Atrium will use OpenGL for rendering, thus (theoretically) supporting all platforms that provide OpenGL API. Our target platforms are Windows (Windows XP and above, 32-bit) and Linux (x86, Ubuntu 8.10 and above). The game will be free (as in freedom) and will be built upon exclusively FLOSS components.

This site is still under construction, stay tuned!


Our first milestone is to create one full-fledged game level as a gameplay demonstration. This will mainly serve as a showcase for our physics engine dmech.

dmech stands for "D mechanics": it is a real-time 3D physics engine written in D language, capable of simulating rigid body dynamics. It features iterative SI solver, high-performance collision detection, all basic geometry shapes and body constraints, arbitrary static trimeshes and ray cast support.

dmech is already usable in game projects (although some functionality is a subject to change). You may check it out on GitHub:
https://github.com/gecko0307/dmech

Other technologies that we use:

  • OpenGL, SDL, FreeType, OpenAL, Ogg/Vorbis
  • DGL as a graphics engine
  • dlib as an utility library
  • Cook2 to automate the building process
  • Probably our own GTK-based level editor
